# Markdown Syntax Quick Guide

Markdown is a lightweight markup language that allows you to format plain text easily. It's widely used for creating web content and documentation. In this quick guide, we'll cover the major Markdown syntax elements.

## Headings

You can create headings using `#` symbols. The number of `#` symbols indicates the heading level:

# Heading 1
## Heading 2
### Heading 3

## Emphasis

You can emphasize text using `*` or `_`:

*Italic* or _Italic_
**Bold** or __Bold__
***Bold and Italic*** or ___Bold and Italic___

## Lists

### Unordered List

- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2

### Ordered List

1. First item
2. Second item
3. Third item

## Links

You can create links like this:

[Visit OpenAI](https://www.openai.com)

## Images

You can embed images using an exclamation mark and square brackets:

![Markdown Logo](https://markdown-here.com/img/icon256.png)

## Blockquotes

> This is a blockquote. You can use it for citing text or adding comments.

## Code

Inline code: `code here`

You can also create code blocks by wrapping the code in triple backticks:

```python
def hello_world():
    print("Hello, world!")
```
